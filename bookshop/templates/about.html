Project overview

About BookNest

BookNest is a practical online bookstore built with Flask, SQLite, HTML, CSS, and JavaScript.

Application features

  • Responsive home page, catalog, details, cart, checkout, and account pages
  • User registration, login, logout, and protected cart/order pages
  • SQLite tables for users, categories, books, cart items, orders, and order items
  • Open Library API data for external covers, authors, ISBNs, and descriptions
  • SEO-friendly titles, meta descriptions, semantic HTML, alt text, and clean URLs

Database relationships

  • One category can contain many books
  • One user can have many cart items and many orders
  • One cart item belongs to one user and one book
  • One order can contain many order items
  • One order item stores the book, quantity, and price at purchase time

Session management

After login, Flask stores the user id and email in a signed session cookie. Protected routes check that session before showing cart, checkout, and order pages. Logout clears the session.

External API

Open Library is useful because the store can enrich local book records with public book metadata such as cover images, authors, publication years, ISBNs, and work descriptions.
